# Unofficial FCU Proof Ganaerator

Just a Document Generator made with .NET and AWS Lambda  

No more Sybase PowerBuilder  
No more Printer  
No more $10  

## Features

- FCU-styled student proof of enrollment
- Customization

## Build

```sh
./tools/build.sh
```
